= EZ Runable Parent

=== EZ 可运行组件构建框架。

== 使用

[source,xml]
----
<parent>
    <groupId>com.ecfront</groupId>
    <artifactId>ez-runnable-parent</artifactId>
    <version>1.2.0</version>
</parent>
----

IMPORTANT: 子项目需要重写`<mainClass></mainClass>`，使用自己的main class

NOTE: 打包输入目录：./target/deploy/

[TIP]
.打包方式
----
. 打成带lib目录的包：`package`
. 只打项目包（不包含依赖）：`jar:jar`
. 打成fat包：`assembly:assembly`
----
