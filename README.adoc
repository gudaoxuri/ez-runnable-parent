= EZ Runable Parent

=== EZ 可运行组件构建框架。

== 使用

[source]
----
<parent>
    <groupId>com.ecfront</groupId>
    <artifactId>ez-runnable-parent</artifactId>
    <version>1.1</version>
</parent>
----
