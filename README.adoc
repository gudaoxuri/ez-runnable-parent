= EZ Runable Parent

=== EZ 可运行组件构建框架。

== 使用

[source,xml]
----
<parent>
    <groupId>com.ecfront</groupId>
    <artifactId>ez-runnable-parent</artifactId>
    <version>1.1</version>
</parent>
----

打包输入目录：./target/deploy/

. 打成带lib目录的包：package
. 只打项目包（不包含依赖）：jar:jar
. 打成fat包：assembly:assembly
